# Econometric analysis in R

This repository contains selected scripts demonstrating econometric methods 
implemented in R as part of my Applied Quantitative Methods coursework.

## Contents

- **OLS and Instrumental Variables**
  - Linear regression models
  - Heteroskedasticity testing (Breusch–Pagan)
  - Robust standard errors
  - Instrumental variable estimation

- **Discrete Choice and Panel Data**
  - Linear Probability Model (LPM)
  - Logit model and marginal effects
  - Fixed Effects and Random Effects models
  - Hausman test

- **Time Series Analysis**
  - Stationarity testing (ADF)
  - ARMA models
  - Basic macroeconomic time series transformations

## Tools

- R  
- Packages: `wooldridge`, `plm`, `lmtest`, `sandwich`, `forecast`
